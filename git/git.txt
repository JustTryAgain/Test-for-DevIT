На линуксе так проще)))
представим что код в директории: ~/work/code
Тогда мы идём в директорию "работа":
cd ~/work
И тут просто клонируем репозиторий которий создал клиент
git clone git@example.com:example/test.git
это создаст директорию пустую test (но там есть файлы гита скрытые, хе-хе)
Теперь ото всё що мы накодили туда копируем
cp -R code/* test/
Теперь переходим в директорию с гитом и нашим только-что скопированным кодом
cd test
Тут проверяем - всё нормально?
git status
если все файлы с кодом показывает добавляем их к будующему коммиту
git add 
и снова проверим всё ли хорошо 
git status
Эта вторая проверка не обязательная, НО я же ответственный)
Должны увидеть те же все файлы только уже в розделе "добавленные к коммиту"
Если всё отлично то коммитим 
git commit -m "Initial commit with code"
А теперь и на сервер можно запушить 
git push origin master
та-да усё теперь самому проверить если есть доступ на сервер, или попросить клиента
